# Ontario Association of Fire Chiefs
> Custom theme by Acro Media Inc.

## Introduction
This custom theme uses the Orange Framework as a base theme.

## Requirements
- [Twig Tweak](https://www.drupal.org/project/twig_tweak)

## Installation
Enable the theme in Drupal.

## Configuration
No configuration needed out of the box.

## Theme Packages

### Theme Styling
Run `npm i` to install the packages.

- To compile: `npm run compile`.
- Watch Sass files for changes and compile: `npm run watch`.

The CSS will be compiled to: `css/*.css`.

### Web Accessibility
You can use `pa11y` to scan URLs for any accessibility issues.
- `npm run pa11y http://oafc.lndo.site/contact-us`.
